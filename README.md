# laravolt/thunderclap

## Installation

Install thunderclap via composer:

```bash
composer require laravolt/thunderclap
```
Add service provider:

```php
Laravolt\Thunderclap\ServiceProvider::class,
```

## Usage

```bash
php artisan clap
```

:clap: clap your hand twice, and follow the magic...
